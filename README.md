# 简介
一个搞笑的句子翻译器。通过将句子拆分成一个个单词的方式，逐单词翻译，最后输出单词拼接的结果，达到“搞笑翻译”的目的。  
## 交互效果
```
翻译哪些句子？（150字以下）：How are you?I'm fine,thank you.And you?
翻译结果
========================================
怎样是你？我是令人满意的，感谢你。和你？

==============================
```
# 快速上手
输入英文句子即可。
